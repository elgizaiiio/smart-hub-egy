# خطة إصلاح صفحة الملفات — تجربة احترافية

## المشاكل الحالية

1. كود HTML يظهر داخل المحادثة بدلاً من البريفيو فقط
2. خطوات البحث تستخدم `Loader2` و `CheckCircle2` بدلاً من النجمة المتحركة (البراند)
3. الهيدر العلوي يحتوي أزرار Chat/Preview — المطلوب زر Preview فقط يظهر بعد نجاح التوليد
4. زر البحث في المستندات موجود ولا حاجة له
5. زر + يفتح `fileInputRef` مباشرة بدون قائمة
6. البريفيو لا يعرض بشكل كامل على الهاتف
7. لا يوجد سكرين شوت تلقائي للملفات المُنشأة

---

## التغييرات

### 1. إصلاح عرض HTML داخل المحادثة

**الملف:** `FilesPage.tsx` (سطر 686-688)

المشكلة: `ReactMarkdown` يعرض كود HTML خام. الحل:

- تنظيف المحتوى من أي HTML قبل العرض بـ regex أقوى
- إذا كان الرد يحتوي HTML، نعرض فقط الـ `---SUMMARY---` في المحادثة
- نقل الـ HTML للبريفيو فقط تلقائياً

### 2. استبدال ResearchFlow بنجمة البراند المتحركة

**الملف:** `ResearchFlow.tsx` — إعادة كتابة كاملة

- استخدام `AnimatedStar` من `CodeStepMessage.tsx` بالضبط (النجمة الدوّارة 8 رؤوس)
- كل خطوة: نجمة متحركة ملونة + نص بجانبها
- الخطوة المكتملة: نجمة ثابتة صغيرة بدون دوران + لون خافت
- الخطوة النشطة: نجمة كبيرة تدور + نص أبيض
- بدون خلفية مربع — خطوات مسطحة مباشرة في المحادثة

### 3. إزالة هيدر Chat/Preview + إضافة زر Preview سفلي

**الملف:** `FilesPage.tsx`

- حذف الـ tab switcher من الهيدر العلوي (سطر 507-510)
- جعل الهيدر شفاف بدون خلفية
- إضافة زر "Preview" عائم في أسفل الشات يظهر فقط بعد نجاح التوليد (`previewHtml !== null`)
- الضغط عليه → يفتح صفحة البريفيو (على الهاتف: `setActiveTab("preview")`)

### 4. إزالة زر البحث من FilePreviewPanel

**الملف:** `FilePreviewPanel.tsx`

- حذف زر Search وشريط البحث بالكامل (سطر 28-44)
- الهيدر: زر رجوع + اسم الملف فقط

### 5. قائمة iOS 26 لزر +

**الملف:** `FilesPage.tsx`

- عند الضغط على `+` (في الإدخال الخارجي أو الداخلي): فتح قائمة منبثقة بتصميم `liquid-glass` ضبابي
- الخيارات:
  - إرفاق ملف
  - إرفاق صورة
  - Google Drive (غير مفعّل — يظهر "قريباً")
  - رفع ملف من الجهاز

### 6. سكرين شوت تلقائي للملفات المُنشأة

**الملف:** `FilesPage.tsx`

- بعد توليد ملف، نعرض بطاقة في المحادثة بدلاً من أزرار عادية:
  - `iframe` مصغّر (thumbnail) للـ HTML المُنشأ بارتفاع ~120px
  - اسم المستند تحته
  - زرين: "تعديل" + "Preview"
  - بتصميم `liquid-glass-subtle` نظيف

### 7. إصلاح البريفيو على الهاتف

**الملف:** `FilePreviewPanel.tsx`

- تغيير الـ iframe ليشمل `<meta name="viewport" content="width=device-width">` في الـ `srcDoc`
- إضافة CSS داخلي للـ iframe يجعل المحتوى متجاوب
- إزالة `p-2 md:p-4` واستخدام `p-0` مع تغطية كاملة

### 8. قوالب ديناميكية (1000+ قالب داخلي)

**الملف:** `FilesPage.tsx` + `filesHtmlBuilders.ts`

- إزالة قالب HTML ثابت واحد
- إضافة تعليمات في system prompt: "اختر تصميماً فريداً يناسب الموضوع من بين أكثر من 1000 نمط: ألوان، خطوط، تخطيطات، أنماط حسب المجال (أكاديمي، تجاري، إبداعي، طبي...)"
- الذكاء الاصطناعي يقرر التصميم تلقائياً حسب السياق

### 9. Timeline/Roadmap/Mindmap — قوالب من قاعدة البيانات

**الملف:** `FilesPage.tsx`

- عند اختيار أي منهم، جلب القوالب من جدول جديد
- عرض القوالب كصور أفقية مثل قوالب Slides الحالية
- ستُضاف من بوت تليجرام
