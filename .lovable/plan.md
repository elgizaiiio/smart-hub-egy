## ترتيب الأخطاء المتوقعة حسب الخطورة

### حرجة (P0) — تمنع الاستخدام:
1. **Computer Use لا يعمل** — مفاتيح HB غير متاحة أو منطق التفعيل معطل
2. **بطء الشات الشديد** — تحميل أدوات غير ضرورية + context كبير
3. **Slides من Canva لا تعمل** — الربط الفعلي غير مكتمل
4. **المكالمات الصوتية لا تعمل** — Deepgram/WebSocket

### عالية (P1) — تؤثر على التجربة:
5. **ThinkingLoader يعرض حالات وهمية** — "يبحث" بدون بحث فعلي
6. **Object Remover يمسح عشوائياً** — mask لا يُرسل بشكل صحيح
7. **LemonData 402** — لا fallback تلقائي
8. **كل الـ imports مباشرة** — لا lazy loading → بطء التحميل الأولي

### متوسطة (P2):
9. **ChatPage.tsx 1038 سطر** — ملف ضخم يصعب صيانته
10. **rememberMe غير مستخدم** — كود ميت
11. **لا React.memo على مكونات الشات** — re-renders غير ضرورية

### منخفضة (P3):
12. **عدم وجود debounce في AnimatedInput**
13. **عدم وجود keyboard shortcuts**
14. **Deep linking محدود**

## التحسينات العامة المقترحة

1. **تقسيم ChatPage.tsx** إلى مكونات أصغر (ChatSidebar, ChatMessages, ChatInput)
2. **Lazy loading لكل الـ routes** عبر `React.lazy` + `Suspense`
3. **React.memo** على ChatMessage, ThinkingLoader, AppSidebar
4. **Error boundaries** على مستوى كل صفحة وليس التطبيق فقط
5. **Retry mechanism** مع exponential backoff لـ API calls
6. **Fallback chain** لمزودي AI: LemonData → Lovable Gateway → deAPI
7. **Health check** دوري لمفاتيح API في `api_keys` table
8. **تقليل حجم System Prompt** للرسائل البسيطة
